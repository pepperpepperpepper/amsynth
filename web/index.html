Tonic split (just-intonation movable key center)

default 12-TET
default linear
Load a non-12-TET scale (e.g. 5-limit JI), then enable the split. Keys below the split point (shown grey) re-root the scale instead of sounding — play one, then play higher keys to hear the tonic move. With the default 12-TET scale the re-root is inaudible but the control zone still swallows those keys.
